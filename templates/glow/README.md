# Glow Bundle Template

A polished Mix & Match bundle template with stronger visual styling, locale-aware labels, richer product card interactions, and configurable section presentation.

## Features

- Styled product cards with quantity controls and secondary image hover support
- Product modal flow for detailed selection and add-ons
- Bundle summary with empty state messaging and progress states
- Configurable section wrapper classes, section background, and spacing
- Configurable locale text for price labels, bundle CTAs, and add-on labels
- Configurable CSS variables for easy theming

## Supported configuration

Currently the template supports the following configuration:

| Feature | Supported |
| ------- | --------- |
| Quantity as option | ✅ |
| Pricing type | Fixed & Dynamic Pricing |
| Add-Ons | ✅ |
| Locale labels | ✅ |
| Section styling controls | ✅ |
