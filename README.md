# 🦊 Foxkit Bundle Templates

This repository contains ready-to-use templates for **Foxsell Bundles**. These templates are designed to help you quickly set up and customize bundle displays on your Shopify store.

## 🚀 How to Use

All available templates can be used by downloading this repository and moving the files to their respective directories in your Shopify theme:

1. **Download the Repository:** Clone or download the ZIP of this repo.
2. **Identify the Template:** Navigate to the `templates/` folder and choose the template you want to use (e.g., `mix-match-fixed-template`).
3. **Move Files:**
   - Upload files from the `assets/` folder to your theme's **Assets** directory.
   - Upload files from the `sections/` folder to your theme's **Sections** directory.
   - Upload files from the `snippets/` folder to your theme's **Snippets** directory.
4. **Configure in Shopify:** Once the files are uploaded, you can start using the sections in your Shopify Theme Editor.
